### 5Ô∏è‚É£ Manually Apply the STIG and Verify the Fix

To manually apply STIG WN11-AU-000500, the Application Event Log maximum size must be configured to 32,768 KB (32 MB) or greater.

This can be accomplished through Group Policy:
- Computer Configuration
- Administrative Templates
- Windows Components
- Event Log Service
- Application
- Specify the maximum log file size (KB)
- Set the value to 32768 or higher

<p align="left">
  <img src="screenshots/Screenshot 2025-12-11 104757.png" width="750">
</p>

Alternatively, the same setting can be enforced through the registry by modifying:
HKLM\SOFTWARE\Policies\Microsoft\Windows\EventLog\Application
Value Name: MaxSize (REG_DWORD)
Value Data: 32768 or greater
